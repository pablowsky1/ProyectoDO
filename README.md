# Ontology Requirements Specification - Group 9

## Project Description
This repository contains the requirement specification documents for an ontology modeling the U.S. film industry. The project focuses on analyzing gender distribution among cast and crew (directors, producers, actors) and correlating this data with Academy Award (Oscar) recognition.

## Datasets Used
1. **Initial Dataset:** U.S. movies with gender-disambiguated actors, directors, and producers.
   - Source: [Figshare](https://figshare.com/articles/dataset/U_S_movies_with_gender-disambiguated_actors_directors_and_producers/4967876)
2. **Extension Dataset:** The Oscar Award, 1927 - 2025.
   - Source: [Kaggle](https://www.kaggle.com/datasets/unanimad/the-oscar-award)
